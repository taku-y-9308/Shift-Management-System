# share_worksheet
毎月21日の0~1時に起動して、編集可能な来月分の勤務表を作成して、「勤務表_各月」に格納する。  
もし、出し忘れた人がいれば手動でこれを起動してもいいですし、スプレッドシート 「シフト回収スクリプト」にボタンがあります。  
手動で、これらのスクリプトを起動した場合には「勤務表ID一覧」の更新を忘れずにお願いします。
